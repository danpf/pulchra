README
======

Description
-----------

Modified PULCHRA 3.07 release for c++ / python library. Turn CA pdbs
into full-atom pdbs

Installation
------------

Requirements
~~~~~~~~~~~~

-  A working c++ compiler (gcc/clang)

::

   pip install pypulchra

Usage
-----

Commandline
~~~~~~~~~~~

::

   pypulchra --input-pdb my.pdb --output-pdb out.pdb

Api
~~~

see ``pypulchra/__init__.py``
